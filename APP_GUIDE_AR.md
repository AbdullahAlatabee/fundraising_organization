# دليل التطبيق الكامل - Fundraising Organization

## نظرة عامة
تطبيق إدارة جمعية خيرية متكامل تم بناؤه باستخدام Flutter مع دعم كامل للعمل دون اتصال بالإنترنت.

## المميزات الرئيسية

### 1. المصادقة والأمان
- تسجيل الدخول والتسجيل
- تشفير كلمات المرور (SHA-256)
- حماية المسارات باستخدام Middleware
- مستخدم افتراضي: `admin@charity.com` / `admin123`

### 2. إدارة الحالات
- إنشاء وعرض حالات التبرع
- تتبع المبالغ المستهدفة والمجمعة
- حالات تلقائية (مفتوحة/مكتملة)
- دعم الصور (محلية أو من الإنترنت)

### 3. إدارة المتبرعين
- قاعدة بيانات كاملة للمتبرعين
- البحث والتصفية
- سجل التبرعات لكل متبرع
- معلومات الاتصال والملاحظات

### 4. إدارة التبرعات
- تسجيل التبرعات (نقدية/عينية)
- ربط التبرعات بالحالات والمتبرعين
- تحديث تلقائي للمبالغ المجمعة
- سجل كامل للتبرعات

### 5. لوحة التحكم
- ملخص مرئي للإحصائيات
- إجمالي التبرعات
- عدد الحالات والمتبرعين
- النشاط الأخير

### 6. الملف الشخصي
- عرض وتعديل معلومات المستخدم
- تحميل صورة الملف الشخصي
- تسجيل الخروج
- الوصول إلى الإعدادات

### 7. التوطين
- دعم كامل للعربية والإنجليزية
- دعم RTL للعربية
- تبديل فوري للغة

### 8. السمات
- وضع فاتح ووضع داكن
- ألوان متسقة (Teal/Blue)
- تصميم Material Design حديث

## البنية التقنية

### التقنيات المستخدمة
- **Flutter**: إطار العمل الأساسي
- **GetX**: إدارة الحالة والتوجيه
- **SQLite**: قاعدة البيانات المحلية
- **SharedPreferences**: حفظ الإعدادات
- **Image Picker**: اختيار الصور

### هيكل المشروع
```
lib/
├── bindings/          # ربط التبعيات
├── controllers/       # منطق الأعمال
├── core/             # الثوابت
├── data/
│   ├── database/     # قاعدة البيانات
│   └── models/       # نماذج البيانات
├── middleware/       # حماية المسارات
├── routes/           # التوجيه
├── themes/           # السمات
├── translations/     # الترجمات
└── views/            # واجهات المستخدم
    ├── auth/         # المصادقة
    ├── cases/        # الحالات
    ├── dashboard/    # لوحة التحكم
    ├── donors/       # المتبرعين
    ├── donations/    # التبرعات
    ├── main/         # الشاشة الرئيسية
    ├── profile/      # الملف الشخصي
    ├── settings/     # الإعدادات
    └── widgets/      # مكونات مشتركة
```

## قاعدة البيانات

### الجداول
1. **users**: معلومات المستخدمين
2. **donors**: بيانات المتبرعين
3. **donation_cases**: حالات التبرع
4. **donations**: سجل التبرعات
5. **images**: مسارات الصور
6. **activity_logs**: سجل النشاطات

### العلاقات
- كل تبرع مرتبط بمتبرع وحالة
- كل حالة/متبرع مرتبط بمستخدم (منشئ)
- سجل النشاطات يتتبع جميع العمليات

## التنقل

### BottomNavigationBar
1. **الرئيسية**: لوحة التحكم
2. **الحالات**: قائمة حالات التبرع
3. **المتبرعين**: قائمة المتبرعين
4. **الملف الشخصي**: معلومات المستخدم

### الميزات الإضافية
- FAB لإضافة حالات/متبرعين
- زر تعديل في الملف الشخصي
- الوصول للإعدادات من الملف الشخصي

## الإصلاحات الأخيرة

### مشاكل GetX
 تم إصلاح "improper use of GetX"
- تحويل Views من `GetView<T>` إلى `StatelessWidget`
- استخدام `Get.find<T>()` بشكل صحيح

### BottomNavigationBar
 يظهر الآن بشكل صحيح
- إزالة Scaffold من Views الفرعية
- الاحتفاظ بـ Scaffold في MainView فقط

### التحسينات
 تحسين الأداء
- تقليل إعادة البناء غير الضرورية
- استخدام Obx بشكل فعال
